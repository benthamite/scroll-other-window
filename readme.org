The only purpose of this repository is to host a copy of Andreas Politz's ~scroll-other-window.el~, so that it can be installed directly via [[https://github.com/radian-software/straight.el][straight]]:

#+begin_src emacs-lisp
(use-package scroll-other-window
  :straight (scroll-other-window
             :host github
	     :repo "benthamite/scroll-other-window")
  :after pdf-tools             
  :demand t)
#+end_src
